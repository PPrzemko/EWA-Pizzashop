# EWA-Pizzashop
Ein einfacher Pizza-Webshop, der mithilfe von HTML, CSS, JavaScript und SQL folgende Funktionen bietet: Bestellungen, Sendungsverfolgung, Übersicht über die Bäcker und Übersicht über die Lieferanten.
